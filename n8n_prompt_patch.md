### Body Battery — поле изменилось

В данных `/all-today` поле `body_battery` теперь содержит:
- `current_level` — актуальный уровень (последнее показание)
- `net_used_since_wake` — израсходовано с момента пробуждения (>0 = потрачено, <0 = восстановлено)
- `charged`, `drained` — суммарные за период (оставлены для справки, не использовать для "израсходовано")

Формат отчёта: "Body Battery: проснулся {wake} → сейчас {current_level} (израсходовано {net_used_since_wake})"
Если net_used_since_wake < 0: "восстановлено {abs(net_used_since_wake)}"
